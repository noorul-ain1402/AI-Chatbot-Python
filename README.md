AI-CHATBOT-PYTHON

AI CHATBOT USING PYTHON

OVERVIEW

This is a simple rule-based AI chatbot developed using Python.

FEATURES

- Handles greetings
- Answers common questions
- Uses dictionaries and if-else logic
- Runs continuously until user exits

TECHNOLOGIES USED

- Python
- Dictionaries
- Loops
- Conditional Statements

HOW TO RUN 

1. Install Python
2. Download chatbot.py
3. Run:

```bash
python chatbot.py
